# BigBoom
RxJava &amp; Retrofit &amp; MVP

这是一个根据[豆瓣电影API](https://developers.douban.com/wiki/?title=movie_v2)所做的电影资讯的App

## 功能说明
* 浏览电影资讯，按类型分为：正在热映、将要上映、TOP250
* 电影、影人资料搜索
* 影人图片、剧照下载
* 预告片播放，弹幕
* 浏览记录保存
* 夜间模式
